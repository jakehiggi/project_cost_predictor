# Amodo Project

## Project Structure

The project is organized into the following directories:

- **app/**: Contains the application code for running the a StreamLit application.
- **data/**: Generates the datasets used in the project.
- **data_exploration/**: Includes the notebook for analyzing and visualizing the data.
- **model/**: Contains machine learning models, training scripts, and related files.

## Getting Started

1. Clone the repository to your local machine.
2. Run the model using either the training data or your own data
3. Run the app script to open an interactive panel.

## Usage

- Use the `data_exploration/` directory to explore and understand the data.
- Train models using scripts in the `model/` directory.
- Run the application from the `app/` directory.
